<h1 align="center">Download Manager</h1>
<p align="center">A simple tool that keeps your download folder clean by asking what to do with each new file.</p>
<br/>
<p>Monitors your download directory and opens a small window asking what to do with new files. You have a few options to choose from, this is what it looks like:</p>
<p align="center">
  <img width="443" height="216" alt="image" src="https://github.com/user-attachments/assets/69143518-beaf-4b3d-874a-540d8c578ded" />
</p>
<p>When you click on <code>Delete Later</code> you can choose one of this options: <code>5 min</code>, <code>10 min</code>, <code>20 min</code>, <code>30 min</code> or <code>1 h</code></p>
<br/>

> [!CAUTION]
> This tool starts automatically when your PC boots.  
> There is no settings menu yet. You must disable it manually (e.g., via Task Manager).

<h1></h1>
<h3>Features:</h3>
<ul>
  <li><strong>Delete Later</strong> – Schedule deletion (e.g., in 5 or 10 minutes, up to 1 hour).</li>
  <li><strong>Move Now</strong> – Move the file to any folder immediately.</li>
  <li><strong>Delete Now</strong> – Instantly remove unwanted or accidental downloads.</li>
  <li><strong>Ignore</strong> – Leave it as it is.</li>
</ul>
